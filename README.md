# 🔐 Password Manager (Version 1)

A simple command-line password manager written in Python.

This project is my first complete Python application and focuses on learning file handling, user input validation, and program organization.

---

## ✨ Features

- ➕ Add new passwords
- 📄 View all saved passwords
- 🔍 Search for passwords by website
- 💾 Store data in a local text file
- 📝 Save additional notes for each account
- 🖥️ Simple command-line interface

---

## 📂 Project Structure

```
Password-Manager/
│
├── main.py          # Main application
├── passwords.txt    # Stored passwords

```

## 🚀 Getting Started

### Requirements

- Python 3.10 or newer

### Clone the repository

```bash
git clone https://github.com/WiredIn-Junior/password-manager.git
```

### Navigate to the project

```bash
cd password-manager
```

### Run the application

```bash
python main.py
```
